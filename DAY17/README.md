### CSS3 Align Justify

텍스트 양쪽 정렬 [`text-align-last`](http://caniuse.com/#search=text-align-last)

![](../Assets/text-align-justify.jpg)

```css
.align-last-justify {
  width: 20rem;
  margin-left: auto;
  margin-right: auto;
  font-size: 6rem;
  text-align: justify;
  -moz-text-align-last: justify;
  text-align-last: justify;
  font-weight: 900;
}
```
